# Regular and irregular verbs

## Regular verbs — adding **-ed**

| Verb ending | Rule | Example |
|-------------|------|---------|
| ends in `-e` | drop `e` + **ed** | live → lived |
| consonant + `-y` | `y` → `i` + **ed** | study → studied |
| vowel + `-y` | keep `y` + **ed** | stay → stayed |
| short vowel + consonant | double consonant + **ed** | stop → stopped |
