# Plural forms of nouns

## Regular rules

| Ending | Rule | Example |
|--------|------|---------|
| consonant or `-e` | + **s** | dog → dogs, name → names |
| `-ss`, `-x`, `-sh`, `-ch`, `-o` | + **es** | kiss → kisses, box → boxes, church → churches, tomato → tomatoes |
| consonant + `-y` | `-y` → **ies** | baby → babies |

## Irregular forms

| Pattern | Examples |
|---------|---------|
| Same form (singular = plural) | deer, sheep, swine |
| `-f` / `-fe` → **ves** | wolf → wolves, wife → wives, knife → knives |
| Vowel change | man → men, woman → women, tooth → teeth, goose → geese, foot → feet, mouse → mice |
| Plural only | trousers, glasses, clothes, spectacles, scissors, goods |
| Singular only | sugar, milk, money, weather, peace |
| Special endings | child → children, ox → oxen |
