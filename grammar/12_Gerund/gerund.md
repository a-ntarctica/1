# Gerund

## Forms

| Form | Structure | Example |
|------|-----------|---------|
| Indefinite (simple) | base + -ing | to ask → asking |
| Perfect | having + past participle | having asked |
| Passive | being + past participle | being asked |

## Functions in sentence

| Function | Example | Translation |
|----------|---------|-------------|
| Subject | **Reading** is useful | Чтение полезно |
| Part of predicate | Your job was **collecting** the material | Сбор материала |
| Subject + predicate (=infinitive) | **Seeing** is **believing** | Увидеть — значит поверить |
| Object | Nobody likes **waiting** | Никто не любит ждать |
| Adverbial | He talked without **stopping** | Говорил без остановки |
| Attribute | I don't see any use in **going** there | Не вижу смысла идти туда |

## Gerund vs Infinitive — key verbs

| Gerund only | Infinitive only | Both (different meaning) |
|-------------|----------------|--------------------------|
| enjoy, mind, finish, avoid, keep, consider | want, hope, decide, plan, refuse | stop, remember, forget, try |

stop + gerund = прекратить (Stop talking!)
stop + infinitive = остановиться чтобы (He stopped to talk.)
