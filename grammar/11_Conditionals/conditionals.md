# Conditionals and Conditional Mood

## Conditional Mood forms

| Form | Structure | Example |
|------|-----------|---------|
| Present Conditional | should/would + bare infinitive | I should/would write |
| Perfect Conditional | should/would + have + past participle | I should/would have written |

## Usage

| Usage | Example | Translation |
|-------|---------|-------------|
| Action depending on condition | If I were you, I would speak about it | На вашем месте я бы поговорил |
| After wish (future desired action) | I wish you would be more polite | Я хотел бы, чтобы ты был вежлив |
| Polite request | I should be too glad if you'd translate this article | Я был бы рад, если бы вы перевели |

## Conditional types (practical)

| Type | Condition | Result | Example |
|------|-----------|--------|---------|
| Type 1 (real) | if + Present Simple | will + infinitive | If it rains, I will stay home |
| Type 2 (unreal present) | if + Past Simple | would + infinitive | If I were you, I would go |
| Type 3 (unreal past) | if + Past Perfect | would have + past participle | If I had known, I would have called |

Note: "If I **were** you" (not "was") — subjunctive mood, formal rule
