# Conjunctions

## Coordinating conjunctions

| Type | Conjunctions | Example |
|------|-------------|---------|
| Connective | and, as well as, both...and, not only...but also, neither...nor | Kate goes to the park, and I go with her |
| Disjunctive | either...or, or | Either go with me or stay here |
| Adversative | but, still, yet | Kate looked for the doll everywhere, but couldn't find it |
| Causal | for | Kate decided to take the umbrella, for it looked like raining |
| Result | therefore, thus, so | I'm not hungry, so there is no need to have dinner |

## Subordinating conjunctions

| Type | Conjunctions | Example |
|------|-------------|---------|
| Time | when, after, before, until/till, while, as, since, as soon as | Let's wait until the rain stops |
| Comparative | as, as...as, as if/as though, than | Say the word as I said it |
| Conditional | if, whether, unless, suppose | He will do it if you ask him |
| Explanatory | that | Kate told me that she would come at seven |
| Causal | because, since, as | I can't go because I'm busy |
| Purpose | so that, in order that, in case | Make a note in case you forget |
| Result | so...that, such...that | I'm so hungry that I can't go anywhere |
| Concessive | though, even if, no matter, whatever | I'll buy the ticket whatever it costs |
